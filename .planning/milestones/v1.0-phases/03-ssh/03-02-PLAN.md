       - `startHost` / `rebuildHost` 在 `PrepareHost` 成功后调用 `WaitForSSHReady`（per D-07）。
       - SSH ready 前不得返回 succeeded（满足 RUNT-03）。
       - 记录阶段事件：`runtime.validating`、`ssh.ready`；失败时保留 `last_error_summary`（per D-12）。
       - 失败保持显式失败，不做自动重试（per D-11）。
  </action>
