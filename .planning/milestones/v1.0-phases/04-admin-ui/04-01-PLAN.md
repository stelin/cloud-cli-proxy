**4. 创建 `admin_auth.go`（新文件）：**

文件位于 `internal/controlplane/http/admin_auth.go`，包含两个导出：

a) `AdminAuthMiddleware(secret []byte) func(http.Handler) http.Handler`：
   - 从 `Authorization` 头提取 `Bearer <token>`
   - 用 `jwt.Parse` + `jwt.WithValidMethods([]string{"HS256"})` 验证
   - key func 中检查 `t.Method.(*jwt.SigningMethodHMAC)` 后返回 `secret`
   - token 无效或过期返回 `writeJSON(w, 401, {"error":"invalid token"})`
   - token 有效调用 `next.ServeHTTP(w, r)`

b) `AdminLoginHandler` struct：
   - 字段：`adminUsername string`, `adminPassword string`, `jwtSecret []byte`, `logger *slog.Logger`
   - 构造函数 `NewAdminLoginHandler(logger, username, password string, jwtSecret []byte)`
   - `ServeHTTP(w, r)` 方法：
     - 仅接受 POST，否则 405
     - 解码 JSON body `{"username":"...","password":"..."}`
     - 用 `hmac.Equal([]byte(req.Password), []byte(h.adminPassword))` 比对密码
     - 用户名或密码不匹配返回 `writeJSON(w, 401, {"error":"invalid credentials"})`
     - 成功时签发 JWT，claims 为 `jwt.RegisteredClaims`：
       - `Subject: "admin"`, `Issuer: "cloud-cli-proxy"`
       - `IssuedAt: jwt.NewNumericDate(time.Now())`
       - `ExpiresAt: jwt.NewNumericDate(time.Now().Add(24 * time.Hour))`
     - 用 `jwt.NewWithClaims(jwt.SigningMethodHS256, claims)` 签名
     - 返回 `writeJSON(w, 200, {"token": tokenStr, "expires_in": 86400})`

导入：`"crypto/hmac"`, `"encoding/json"`, `"fmt"`, `"log/slog"`, `nethttp "net/http"`, `"strings"`, `"time"`, `"github.com/golang-jwt/jwt/v5"`

**步骤 10：创建 `src/lib/api.ts`**
```typescript
const API_BASE = "/v1/admin";

export class ApiError extends Error {
  constructor(public status: number, message: string) {
    super(message);
  }
}

export async function apiFetch<T>(path: string, init?: RequestInit): Promise<T> {
  const token = localStorage.getItem("admin_token");
  const res = await fetch(`${API_BASE}${path}`, {
    ...init,
    headers: {
      "Content-Type": "application/json",
      ...(token ? { Authorization: `Bearer ${token}` } : {}),
      ...init?.headers,
    },
  });

  if (res.status === 401) {
    localStorage.removeItem("admin_token");
    window.location.href = "/login";
    throw new ApiError(401, "Unauthorized");
  }

  if (!res.ok) {
    const body = await res.text();
    throw new ApiError(res.status, body);
  }

  return res.json();
}
```

);
