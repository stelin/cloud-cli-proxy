**2. 在 `queries.go` 中新增 5 个 Repository 方法：**

a) `GetUser(ctx, userID string) (User, error)`：
```sql
SELECT id::text, username, status, created_at, updated_at
FROM users WHERE id = $1
```
扫描 5 字段到 User struct。如 `pgx.ErrNoRows` 返回 wrapped error。

b) `CreateUser(ctx, params CreateUserParams) (User, error)`：
```sql
INSERT INTO users (username, password_hash, status)
VALUES ($1, $2, 'active')
RETURNING id::text, username, status, created_at, updated_at
```
扫描到 User struct。username 唯一约束冲突应返回可区分的错误。

c) `UpdateUserStatus(ctx, userID string, status string) (User, error)`：
```sql
UPDATE users SET status = $2, updated_at = NOW() WHERE id = $1
RETURNING id::text, username, status, created_at, updated_at
```
status 值限定为 `active` 或 `disabled`。

d) `DeleteUser(ctx, userID string) error`：
```sql
DELETE FROM users WHERE id = $1
```
数据库 CASCADE 会自动清理关联 hosts 和 bindings。如影响行数为 0 返回 not found 错误。

e) `UpdateUserPassword(ctx, userID string, passwordHash string) error`：
```sql
UPDATE users SET password_hash = $1, updated_at = NOW() WHERE id = $2
```

f) `ListHostsByUserID(ctx, userID string) ([]Host, error)`：
```sql
SELECT id::text, user_id::text, status, template_image_ref, home_volume_name, slot_key, created_at, updated_at
FROM hosts WHERE user_id = $1
ORDER BY created_at ASC
```
用于用户详情页展示该用户的主机列表。

**3. 创建 `admin_users.go`（新文件）：**

定义接口：
```go
type AdminUserStore interface {
    ListUsers(context.Context) ([]repository.User, error)
    GetUser(context.Context, string) (repository.User, error)
    CreateUser(context.Context, repository.CreateUserParams) (repository.User, error)
    UpdateUserStatus(context.Context, string, string) (repository.User, error)
    DeleteUser(context.Context, string) error
    UpdateUserPassword(context.Context, string, string) error
    ListHostsByUserID(context.Context, string) ([]repository.Host, error)
}
```

`AdminUsersHandler` struct：
- 字段：`logger *slog.Logger`, `store AdminUserStore`
- 构造函数 `NewAdminUsersHandler(logger, store)`

路由 handler 方法（每个返回 `http.Handler`）：

a) `List() http.Handler` — `GET /v1/admin/users`：
   - 调用 `store.ListUsers`
   - 返回 `{"users": [...]}`

b) `Get() http.Handler` — `GET /v1/admin/users/{userID}`：
   - `userID := r.PathValue("userID")`
   - 调用 `store.GetUser` + `store.ListHostsByUserID`
   - 返回 `{"user": {...}, "hosts": [...]}`
   - 404 if not found

c) `Create() http.Handler` — `POST /v1/admin/users`：
   - 解码 body `{"username": "...", "password": "..."}`
   - 校验：username 非空 + 长度 3-50，password 非空 + 长度 >= 8
   - 用 `bcrypt.GenerateFromPassword([]byte(password), bcrypt.DefaultCost)` 哈希密码
   - 调用 `store.CreateUser`
   - 返回 201 `{"user": {...}}`
   - username 冲突返回 409 `{"error": "username already exists"}`

d) `UpdateStatus() http.Handler` — `PATCH /v1/admin/users/{userID}`：
   - 解码 body `{"status": "active"|"disabled"}`
   - 校验 status 值只能是 `active` 或 `disabled`
   - 调用 `store.UpdateUserStatus`（per D-10：禁用用户不影响已运行主机）
   - 返回 `{"user": {...}}`
   - 404 if not found

e) `Delete() http.Handler` — `DELETE /v1/admin/users/{userID}`：
   - `userID := r.PathValue("userID")`
   - 调用 `store.DeleteUser`（DB CASCADE 处理关联数据，per D-08）
   - 返回 204 No Content
   - 404 if not found

f) `RotatePassword() http.Handler` — `POST /v1/admin/users/{userID}/rotate-password`（per D-09）：
   - 用 `crypto/rand` 生成 20 字符随机密码（大写 + 小写 + 数字 + 特殊字符）：
     ```go
     const charset = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#$%&*"
     b := make([]byte, 20)
     for i := range b {
         n, _ := rand.Int(rand.Reader, big.NewInt(int64(len(charset))))
         b[i] = charset[n.Int64()]
     }
     newPassword := string(b)
     ```
   - 用 bcrypt 哈希后调用 `store.UpdateUserPassword`
   - 返回 `{"new_password": "..."}`（一次性展示，前端提供复制按钮）

导入：`"crypto/rand"`, `"math/big"`, `"encoding/json"`, `"errors"`, `"log/slog"`, `nethttp "net/http"`, `"github.com/jackc/pgx/v5"`, `"golang.org/x/crypto/bcrypt"`, `"github.com/zaneliu/cloud-cli-proxy/internal/store/repository"`

**1. 创建 `src/hooks/use-users.ts`** — TanStack Query hooks：

```typescript
import { useQuery, useMutation, useQueryClient } from "@tanstack/react-query";
import { apiFetch } from "@/lib/api";

interface User {
  id: string;
  username: string;
  status: string;
  created_at: string;
  updated_at: string;
}

interface Host {
  id: string;
  user_id: string;
  status: string;
  template_image_ref: string;
  slot_key: string;
  created_at: string;
  updated_at: string;
}

export function useUsers() {
  return useQuery({
    queryKey: ["users"],
    queryFn: () => apiFetch<{ users: User[] }>("/users"),
  });
}

export function useUser(userId: string) {
  return useQuery({
    queryKey: ["users", userId],
    queryFn: () => apiFetch<{ user: User; hosts: Host[] }>(`/users/${userId}`),
  });
}

export function useCreateUser() {
  const qc = useQueryClient();
  return useMutation({
    mutationFn: (data: { username: string; password: string }) =>
      apiFetch<{ user: User }>("/users", { method: "POST", body: JSON.stringify(data) }),
    onSuccess: () => qc.invalidateQueries({ queryKey: ["users"] }),
  });
}

export function useUpdateUserStatus() {
  const qc = useQueryClient();
  return useMutation({
    mutationFn: ({ userId, status }: { userId: string; status: string }) =>
      apiFetch<{ user: User }>(`/users/${userId}`, { method: "PATCH", body: JSON.stringify({ status }) }),
    onSuccess: () => qc.invalidateQueries({ queryKey: ["users"] }),
  });
}

export function useDeleteUser() {
  const qc = useQueryClient();
  return useMutation({
    mutationFn: (userId: string) =>
      apiFetch(`/users/${userId}`, { method: "DELETE" }),
    onSuccess: () => qc.invalidateQueries({ queryKey: ["users"] }),
  });
}

export function useRotatePassword() {
  return useMutation({
    mutationFn: (userId: string) =>
      apiFetch<{ new_password: string }>(`/users/${userId}/rotate-password`, { method: "POST" }),
  });
}
```
