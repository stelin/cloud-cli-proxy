**1. 创建 `internal/controlplane/scheduler/scheduler.go`（per D-02, D-17）：**

```go
package scheduler

import (
    "context"
    "log/slog"
    "sync"
    "time"
)

type Job struct {
    Name     string
    Interval time.Duration
    Fn       func(ctx context.Context) error
}

type Scheduler struct {
    logger *slog.Logger
    jobs   []Job
}

func New(logger *slog.Logger, jobs []Job) *Scheduler {
    return &Scheduler{logger: logger, jobs: jobs}
}

func (s *Scheduler) Run(ctx context.Context) {
    var wg sync.WaitGroup
    for _, job := range s.jobs {
        wg.Add(1)
        go func(j Job) {
            defer wg.Done()
            ticker := time.NewTicker(j.Interval)
            defer ticker.Stop()
            s.logger.Info("scheduler job started", "job", j.Name, "interval", j.Interval)
            for {
                select {
                case <-ctx.Done():
                    s.logger.Info("scheduler job stopping", "job", j.Name)
                    return
                case <-ticker.C:
                    if err := j.Fn(ctx); err != nil {
                        s.logger.Error("scheduler job failed", "job", j.Name, "error", err)
                    }
                }
            }
        }(job)
    }
    wg.Wait()
}
```

```go
package scheduler

import (
    "context"
    "log/slog"
    "github.com/zaneliu/cloud-cli-proxy/internal/agentapi"
    "github.com/zaneliu/cloud-cli-proxy/internal/store/repository"
)

type ExpiryStore interface {
    ListExpiredActiveUsers(context.Context) ([]repository.User, error)
    UpdateUserStatus(context.Context, string, string) (repository.User, error)
    ListRunningHostsByUserID(context.Context, string) ([]repository.Host, error)
    RecordEvent(context.Context, repository.RecordEventParams) (repository.Event, error)
}

type HostActionQueuer interface {
    QueueHostAction(context.Context, string, agentapi.HostAction, string) (repository.Task, error)
}

type ExpiryScanner struct {
    logger *slog.Logger
    store  ExpiryStore
    queue  HostActionQueuer
}

func NewExpiryScanner(logger *slog.Logger, store ExpiryStore, queue HostActionQueuer) *ExpiryScanner {
    return &ExpiryScanner{logger: logger, store: store, queue: queue}
}
```

`Scan` 方法逻辑（per D-02, D-05, D-06, D-07）：
1. 调用 `store.ListExpiredActiveUsers(ctx)` 获取待过期用户列表
2. 对每个用户：
   a. 调用 `store.UpdateUserStatus(ctx, user.ID, "expired")`，SQL 带 `WHERE status = 'active'` 保证幂等（per RESEARCH Pitfall 2）
   b. 调用 `store.RecordEvent(ctx, ...)` 记录 `user.expired` 事件，metadata 包含 `{"operator": "system", "reason": "expires_at reached", "expires_at": user.ExpiresAt}`，UserID 设为 `&user.ID`
   c. 调用 `store.ListRunningHostsByUserID(ctx, user.ID)` 获取运行中主机
   d. 对每个 running 主机：调用 `queue.QueueHostAction(ctx, host.ID, agentapi.ActionStopHost, "system:expiry")`，然后 `store.RecordEvent(ctx, ...)` 记录 `host.stop.expired` 事件（per D-07），metadata 包含 `{"operator": "system", "user_id": user.ID, "host_id": host.ID, "reason": "user expired"}`，HostID 设为 `&host.ID`，UserID 设为 `&user.ID`
3. 如果 `ListExpiredActiveUsers` 返回空切片，直接返回 nil（无日志）
4. 如果有用户被标记过期，用 Info 级别日志记录 `"expiry scan completed", "expired_count", len(expiredUsers)`
5. 单个用户处理失败时，记录 Error 日志但继续处理下一个（不中断整个扫描）
