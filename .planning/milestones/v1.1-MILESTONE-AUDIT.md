## 技术债务

### Phase 8: SingBoxProvider 与受管镜像

- `stopHost` 不调用 `CleanupHost`，仅 `rebuildHost` 路径显式清理 sing-box 进程和 veth。容器停止时 sing-box 随 PID namespace 自动退出，但宿主机侧 mgmt veth 可能遗留。

### Phase 9: 前端适配与代理测试

- 代理测试 API 在控制面进程内探测，vmess/ss/trojan 协议依赖控制面环境安装 sing-box 二进制（当前仅受管用户镜像预装），默认容器化部署下这三种协议的测试可能失败
- 测试结果仅存于前端会话 Map 中，页面刷新后丢失，未持久化到后端
- WireGuard 类型出口 IP 的测试按钮存在但无法执行有效探测
