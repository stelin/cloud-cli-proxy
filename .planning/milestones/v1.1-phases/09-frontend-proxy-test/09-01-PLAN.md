**3. getProxyDialer function** — `func getProxyDialer(ctx context.Context, proxyConfig json.RawMessage) (dialer contextDialer, cleanup func(), err error)`:
- Parse proxy_config into `map[string]any`
- Extract `type` (string), `server` (string), `server_port` (float64→int)
- Switch on type:
  - **"socks"**: Extract username/password if present. Build `proxy.Auth` if username non-empty. Call `proxy.SOCKS5("tcp", fmt.Sprintf("%s:%d", server, port), auth, proxy.Direct)`. Assert `proxy.ContextDialer` and return. No cleanup needed.
  - **"http"**: Build `*url.URL{Scheme: "http", Host: fmt.Sprintf("%s:%d", server, port)}`. If TLS config exists in proxy_config (key "tls" with "enabled": true), use scheme "https". If username/password present, set `url.URL.User = url.UserPassword(username, password)`. Return an `httpProxyDialer` wrapper that creates `net.Dialer` and dials through HTTP CONNECT. No cleanup needed.
  - **"vmess", "shadowsocks", "trojan"**: Call `startLocalSingBox(ctx, proxyConfig)` to get localPort + cleanup. Then use `proxy.SOCKS5("tcp", fmt.Sprintf("127.0.0.1:%d", localPort), nil, proxy.Direct)` to get a dialer to the local sing-box SOCKS5 inbound. Return dialer + cleanup.

**4. httpProxyDialer struct** — wraps `*url.URL` to implement contextDialer:
```go
type httpProxyDialer struct {
    proxyURL *url.URL
}

func (d *httpProxyDialer) DialContext(ctx context.Context, network, addr string) (net.Conn, error) {
    // Connect to proxy server
    proxyAddr := d.proxyURL.Host
    conn, err := (&net.Dialer{}).DialContext(ctx, "tcp", proxyAddr)
    if err != nil {
        return nil, fmt.Errorf("connect to HTTP proxy: %w", err)
    }
    // Send CONNECT request
    connectReq := fmt.Sprintf("CONNECT %s HTTP/1.1\r\nHost: %s\r\n", addr, addr)
    if d.proxyURL.User != nil {
        auth := base64.StdEncoding.EncodeToString([]byte(d.proxyURL.User.String()))
        connectReq += fmt.Sprintf("Proxy-Authorization: Basic %s\r\n", auth)
    }
    connectReq += "\r\n"
    if _, err := conn.Write([]byte(connectReq)); err != nil {
        conn.Close()
        return nil, fmt.Errorf("send CONNECT: %w", err)
    }
    // Read response
    br := bufio.NewReader(conn)
    resp, err := nethttp.ReadResponse(br, nil)
    if err != nil {
        conn.Close()
        return nil, fmt.Errorf("read CONNECT response: %w", err)
    }
    if resp.StatusCode != 200 {
        conn.Close()
        return nil, fmt.Errorf("HTTP proxy CONNECT failed: %s", resp.Status)
    }
    return conn, nil
}
```

- If actual is non-empty but different: return `{Status: "fail", Expected: expectedIP, Actual: actual, Sources: sources}`
- If no source returned a result: return `{Status: "error", Expected: expectedIP, Error: "all IP detection services unavailable"}`

**8. testDNSLeak function** — `func testDNSLeak(ctx context.Context, client *nethttp.Client) DNSLeakCheckResult`:
- Read host DNS servers: parse `/etc/resolv.conf` for lines starting with "nameserver", collect IPs into `hostDNS []string`
- Through the proxy client, GET `https://ipleak.net/json/`. Parse JSON response. Extract `"ip"` field as the proxy-visible IP.
- If the request succeeds, DNS resolution through the proxy path worked (the SOCKS5 dialer resolves domains remotely). Report `{Status: "pass", DNSServersDetected: hostDNS, LocalDNSLeaked: false}`.

- If ipleak.net returns an IP that matches the host's public IP (not the egress IP), this suggests DNS or routing is not going through the proxy: return `{Status: "fail", LocalDNSLeaked: true, DNSServersDetected: hostDNS}`.

```go
return nethttp.HandlerFunc(func(w nethttp.ResponseWriter, r *nethttp.Request) {
    ctx, cancel := context.WithTimeout(r.Context(), 30*time.Second)
    defer cancel()

    ipID := r.PathValue("ipID")
    ip, err := h.store.GetEgressIP(ctx, ipID)
    if err != nil {
        if errors.Is(err, pgx.ErrNoRows) {
            writeJSON(w, nethttp.StatusNotFound, map[string]string{"error": "egress ip not found"})
            return
        }
        h.logger.Error("get egress ip for test failed", "ip_id", ipID, "error", err)
        writeJSON(w, nethttp.StatusInternalServerError, map[string]string{"error": "get egress ip failed"})
        return
    }

    if ip.TunnelType != network.TunnelTypeProxy {
        writeJSON(w, nethttp.StatusOK, ProbeResult{
            Status:   "error",
            TestedAt: time.Now().UTC(),
            Message:  "WireGuard 类型出口 IP 在容器启动时自动验证，不支持手动测试",
        })
        return
    }

    if ip.ProxyConfig == nil {
        writeJSON(w, nethttp.StatusBadRequest, map[string]string{"error": "proxy_config is empty"})
        return
    }

    dialer, cleanup, err := getProxyDialer(ctx, ip.ProxyConfig)
    if cleanup != nil {
        defer cleanup()
    }
    if err != nil {
        writeJSON(w, nethttp.StatusOK, ProbeResult{
            Status:   "error",
            TestedAt: time.Now().UTC(),
            Message:  fmt.Sprintf("无法建立代理连接: %v", err),
        })
        return
    }

    httpClient := &nethttp.Client{
        Transport: &nethttp.Transport{
            DialContext:       dialer.DialContext,
            DisableKeepAlives: true,
        },
        Timeout: 25 * time.Second,
    }

    result := ProbeResult{TestedAt: time.Now().UTC()}
    result.Results.Connectivity = testConnectivity(ctx, httpClient)
    result.Results.EgressIP = testEgressIPMatch(ctx, httpClient, ip.IPAddress)
    result.Results.DNSLeak = testDNSLeak(ctx, httpClient)

    // Compute overall status
    statuses := []string{
        result.Results.Connectivity.Status,
        result.Results.EgressIP.Status,
        result.Results.DNSLeak.Status,
    }
    passCount := 0
    for _, s := range statuses {
        if s == "pass" {
            passCount++
        }
    }
    switch {
    case passCount == 3:
        result.Status = "passed"
    case passCount > 0:
        result.Status = "partial"
    default:
        result.Status = "failed"
    }

    writeJSON(w, nethttp.StatusOK, result)
})
```
