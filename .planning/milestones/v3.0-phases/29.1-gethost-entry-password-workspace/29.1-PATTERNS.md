### 1. `internal/store/repository/queries.go`（`GetHost` + 5 个 `List*` 修补 + SQL 常量抽取）

**Role:** repository / SQL — **Data Flow:** request-response (SELECT `hosts`)

**主要形态类比：`getHostByShortIDSQL`（包级 `const` SQL + 回归测试可 import）**

```1152:1173:internal/store/repository/queries.go
// getHostByShortIDSQL 将 SQL 文本提升为包级常量，方便数据层回归测试断言
// Wave 2 所需列（template_image_ref）已纳入 SELECT，且保持对 sshproxy / entry 的向后兼容。
const getHostByShortIDSQL = `
	SELECT h.id::text, h.short_id, h.entry_password, h.status,
	       h.user_id::text, u.status, u.username,
	       COALESCE(h.template_image_ref, '')
	FROM hosts h
	JOIN users u ON u.id = h.user_id
	WHERE h.short_id = $1
`

func (r *Repository) GetHostByShortID(ctx context.Context, shortID string) (HostSSHAuth, error) {
	var item HostSSHAuth
	if err := r.db.QueryRow(ctx, getHostByShortIDSQL, shortID).Scan(
		&item.HostID, &item.HostShortID, &item.EntryPassword,
		&item.HostStatus, &item.UserID, &item.UserStatus, &item.Username,
		&item.TemplateImageRef,
	); err != nil {
		return HostSSHAuth{}, fmt.Errorf("get host by short_id: %w", err)
	}
	return item, nil
}
```

**字段顺序类比（SELECT 列序 + Scan 列序一一对齐）：`ListHostsByUserID`**

**供 planner 直接复制的做法：**

1. 在每个目标函数之前增加包级 `const *SQL = \`...\`` 文本（仿 `getHostByShortIDSQL`）。
2. 在现有 `SELECT` 列序的 `COALESCE(short_id, '')` 之后紧跟插入 `COALESCE(entry_password, '')`（models.go 第 5 位字段对齐）。
3. `Scan` 列表在 `&item.ShortID` 之后紧跟 `&item.EntryPassword`。
4. 6 个函数需一次整批处理（避免 §1.3 中任一回归漏列再次踩坑）。
5. 不动 `WHERE / ORDER BY` / 其他列，diff 仅 "加 1 列 + 抽常量"。

**错误处理 / 返回语义：** 保留现有 `fmt.Errorf("... : %w", err)` 包装格式，不调整。

**Planner 应直接在 `chpasswd` 下一行插入的片段（RESEARCH §3 已给出完整形态）：**

```bash
# 验证密码已被成功设置，避免历史"密码退化为 workspace"的静默失败复现。
# passwd -S 第 2 列：P / PS = 正常设置密码，L / LK = 锁定，NP = 无密码。
status="$(passwd -S "${RUN_USER}" 2>/dev/null | awk '{print $2}' || echo UNSET)"
case "$status" in
  P|PS) ;;
  *)
    echo "[entrypoint] FATAL: password status for ${RUN_USER} is '${status}', refusing to start" >&2
    exit 1
    ;;
esac
```

- 只在 `chpasswd` 行之后、`sysctl -w net.ipv6.conf.all.disable_ipv6=1` 之前插入，不动其他段（避免污染提交范围）。
