**local.go** — LocalManager 核心结构体:
```go
type LocalOptions struct {
    ProjectDir    string // 当前项目目录（用于容器名哈希和 workspace mount）
    Port          int    // 0 = 自动分配
    EgressConfig  string // --egress-config 文件路径，空 = 不启用
    ImageName     string // 默认从 image.lock 读取
    MemoryLimitMB int    // 默认 4096
    CPULimit      float64 // 默认 2.0
}

type SSHInfo struct {
    Host     string
    Port     string
    User     string
    Password string
}

type LocalManager struct {
    opts       LocalOptions
    dockerFunc func(ctx context.Context, args ...string) ([]byte, error) // 可测试注入
}
```

`Up(ctx) (SSHInfo, error)` 方法流程:
1. 确保镜像可用（`docker pull` 或检查本地）
2. 生成随机密码（GeneratePassword）
3. 计算容器名（localContainerName）
4. 如果同名容器已存在，stop + rm
5. 构建 docker create 参数
6. 执行 docker create + docker start
7. 提取 SSH 端口（inspectSSHPort）
8. 返回 SSHInfo

`Down(ctx) error` 方法：stop + rm 容器（Plan 02 会复用）

`Status(ctx) (*ContainerStatus, error)` 方法：inspect 容器（Plan 02 会复用）
