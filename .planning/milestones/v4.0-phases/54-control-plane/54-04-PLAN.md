### T5 — `scripts/uat-bypass-fixture-up.sh` 收敛

文件：`scripts/uat-bypass-fixture-up.sh`

v3.5 bypass fixture 用 sing-box-gateway 镜像起一个旁路 stub 容器（L371-405）。v4.0 单容器化后该 fixture 仍可保留 v3.5 测试场景（uat-bypass 是 v3.5 落地物），但 gateway 镜像不再 build；脚本中 `GATEWAY_IMAGE` 默认值降级到 alpine：

把：
```bash
GATEWAY_IMAGE="${GATEWAY_IMAGE:-cloud-cli-proxy-sing-gateway:local}"
...
if docker image inspect "$GATEWAY_IMAGE" >/dev/null 2>&1; then
  info "复用 gateway 镜像：$GATEWAY_IMAGE"
else
  warn "gateway 镜像 $GATEWAY_IMAGE 不存在，退化到 alpine 占位（I5/I8 走 lenient）"
  GATEWAY_IMAGE="alpine:3.20"
fi
```

改为：
```bash
# v4.0 (Phase 54): sing-box-gateway 镜像已退役；v3.5 bypass fixture 仍可跑，
# 但不再尝试拉 gateway 镜像，直接用 alpine 占位（I5/I8 lenient 模式）。
GATEWAY_IMAGE="${GATEWAY_IMAGE:-alpine:3.20}"
info "v4.0 单容器化：bypass fixture 使用 alpine 占位（GATEWAY_IMAGE=$GATEWAY_IMAGE）"
```

保留 env var override 接口，但默认值改为 alpine。
