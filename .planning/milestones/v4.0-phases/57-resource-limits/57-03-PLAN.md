```typescript
export interface ResourceLimitsValue {

}
```

**预设值定义（来自 D-05）：**

内存预设（单位 MB）：`[无限制, 1024, 2048, 4096, 8192, 16384]`，默认 4096
- 映射：`{ label: "无限制", value: 0 }` / `{ label: "1 GB", value: 1024 }` / `{ label: "2 GB", value: 2048 }` / `{ label: "4 GB (默认)", value: 4096 }` / `{ label: "8 GB", value: 8192 }` / `{ label: "16 GB", value: 16384 }` / `{ label: "自定义...", value: -1 }`

CPU 预设：`[无限制, 0.5, 1, 2, 4, 8]`，默认 2.0
- 映射：`{ label: "无限制", value: 0 }` / `{ label: "0.5 核", value: 0.5 }` / `{ label: "1 核", value: 1 }` / `{ label: "2 核 (默认)", value: 2 }` / `{ label: "4 核", value: 4 }` / `{ label: "8 核", value: 8 }` / `{ label: "自定义...", value: -1 }`

磁盘预设（单位 GB）：`[无限制, 10, 20, 50, 100]`，默认 20
- 映射：`{ label: "无限制", value: 0 }` / `{ label: "10 GB", value: 10 }` / `{ label: "20 GB (默认)", value: 20 }` / `{ label: "50 GB", value: 50 }` / `{ label: "100 GB", value: 100 }` / `{ label: "自定义...", value: -1 }`

**组件行为：**
- 每个选择器渲染为一个 `div` 容器，内含 `Label` + `Select` + 条件渲染的 `Input`（当选择"自定义"时）
- `value` 为 `null` 时视为"默认"（对应创建时不传参数的场景），显示"默认 (4 GB)"类标签
- 选择预设值时直接调用 `onChange` 更新该字段
- 选择"自定义"时显示数字输入框，onChange 在 blur 或回车时触发
- 内存输入框后缀 "MB"，CPU 输入框后缀 "核"，磁盘输入框后缀 "GB"
- `disabled=true` 时 Select 和 Input 均 disabled

```typescript
export function usePatchHostResources(hostId: string) {
  const qc = useQueryClient();
  return useMutation({
    mutationFn: (data: {
      memory_limit_mb?: number | null;
      cpu_limit?: number | null;
      disk_limit_gb?: number | null;
    }) =>
      apiFetch(`/hosts/${hostId}/resources`, {
        method: "PATCH",
        body: JSON.stringify(data),
      }),
    onSuccess: () => {
      qc.invalidateQueries({ queryKey: ["hosts", hostId] });
      toast.success("资源限制已更新");
    },
    onError: (err: Error) => {
      toast.error(err.message || "更新资源限制失败");
    },
  });
}
```
