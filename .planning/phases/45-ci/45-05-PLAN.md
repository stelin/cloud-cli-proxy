<lint_script_design>
<!-- scripts/lint-no-bare-sleep.sh 目标 -->

行为：
  - 仓库根目录运行
  - 找出 tests/e2e/ 下所有 *.go（含子目录）中匹配 ^\s*time\.Sleep\( 的行
  - 注释行（^\s*//）排除（用 grep -v 或 -E 否定）
  - 命中即打印「<file>:<line>: <content>」并 exit 1
  - 无命中 → exit 0
  - --help → 中文 usage + 退出码说明 → exit 0
  - 在 tests/e2e/ 目录不存在时 → 打印 warn 行 → exit 0（Phase 45 之前的分支兼容）

参考实现：
  #!/usr/bin/env bash
  set -euo pipefail

  TARGET_DIR="${TARGET_DIR:-tests/e2e}"
  SCRIPT_NAME="$(basename "$0")"

  usage() {
      cat <<EOF
  $SCRIPT_NAME — Cloud CLI Proxy e2e 套件「禁止裸 sleep」守护

  用途：
      扫描 \$TARGET_DIR（默认 tests/e2e）下所有 *.go 中 time.Sleep( 调用。
      命中即认为违反 v3.6 Phase 45 E2E-05 决策（必须改用 harness.WaitFor 系列 helper）。

  用法：
      bash $SCRIPT_NAME              # 扫描 tests/e2e/
      TARGET_DIR=tests/leak bash $SCRIPT_NAME
      bash $SCRIPT_NAME --help       # 显示本帮助

  退出码：
      0  无命中（或目标目录不存在 — 打印 warn 后退出）
      1  发现裸 sleep
  EOF
  }

  case "${1:-}" in
      -h|--help) usage; exit 0 ;;
  esac

  if [[ ! -d "$TARGET_DIR" ]]; then
      echo "[warn] target dir not found: $TARGET_DIR (skipped)"
      exit 0
  fi

  matches="$(grep -RInE --include='*.go' '^\s*time\.Sleep\(' "$TARGET_DIR" || true)"
  if [[ -n "$matches" ]]; then
      echo "[fail] 发现裸 time.Sleep 调用（违反 Phase 45 E2E-05）："
      echo "$matches"
      echo
      echo "请改用 tests/e2e/harness/waitfor.go 提供的 WaitFor / WaitForLog / WaitForPort / WaitForHTTP / WaitForExec 系列。"
      exit 1
  fi

  echo "[ok] $TARGET_DIR 内无裸 time.Sleep"
  exit 0

关键：
  - 严禁用 `grep -c` 然后比对 == 0（comment 行可能也匹配） — 这里 grep -RIn 直接列出行号，命中即 fail
  - `--include='*.go'` 仅扫描 Go 源码
  - 注释行：当前正则 `^\s*time\.Sleep\(` 不会匹配 `// time.Sleep(`（因为前导是 `//`，不会先空白再立刻 `time.Sleep`）；如担心，可加 `grep -v '^\s*//'` 后置过滤
</lint_script_design>
