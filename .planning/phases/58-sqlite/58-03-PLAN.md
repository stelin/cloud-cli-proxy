**Part A: 重写 queries_bypass.go（652 行）**

应用与 queries.go 完全相同的替换规则（per D-10, D-11, D-12, D-13）：

1. **导入变更：**
   - 移除 `github.com/jackc/pgx/v5`
   - 添加 `database/sql` 和 `github.com/google/uuid`

2. **scan 辅助函数签名变更：**
   - `scanBypassPreset(row pgx.Row, out *BypassPreset) error` → 使用 `*sql.Row` 或自定义 scanner 接口
   - `scanBypassRule(row pgx.Row, out *BypassRule) error` → 同上
   - `scanBypassBinding(row pgx.Row, out *BypassBinding) error` → 同上
   - `scanBypassSnapshot(row pgx.Row, out *BypassSnapshot) error` → 同上
   - 注意：`pgx.Row` 是 interface（QueryRow 和 Rows 都实现 Scan），database/sql 的 `*sql.Row` 和 `*sql.Rows` 是不同类型
   - 推荐方案：将 scan 函数改为接受 `interface{ Scan(dest ...any) error }` 或分别为 QueryRow 和 Query 场景编写

3. **查询调用替换：**
   - `r.db.Query(ctx, ...)` → `r.db.QueryContext(ctx, ...)`
   - `r.db.QueryRow(ctx, ...)` → `r.db.QueryRowContext(ctx, ...)`
   - `r.db.Exec(ctx, ...)` → `r.db.ExecContext(ctx, ...)`

4. **占位符和类型转换：** $N → ?，id::text → id

5. **错误替换：** pgx.ErrNoRows → sql.ErrNoRows

6. **UUID 生成：** INSERT 中缺少 id 的添加 uuid.NewString()

3. **New() 函数重写（per D-08, D-09）：**
   ```go
   db, err := sql.Open("sqlite", cfg.DatabaseURL)
   // DSN 格式：file:/path/to/cloud-cli-proxy.db 或 :memory: 用于测试
   // 对于 file: DSN，添加 _journal_mode=WAL&_busy_timeout=5000&_foreign_keys=ON
   // 或使用 PRAGMA 语句在连接建立后执行

   // 方案 A（推荐）：通过 DSN 参数设置
   // dsn := cfg.DatabaseURL + "?_journal_mode=WAL&_busy_timeout=5000&_foreign_keys=ON"

   // 方案 B：通过 PRAGMA 语句设置
   for _, pragma := range []string{
       "PRAGMA journal_mode=WAL",
       "PRAGMA foreign_keys=ON",
       "PRAGMA busy_timeout=5000",
   } {
       if _, err := db.ExecContext(ctx, pragma); err != nil {
           return nil, fmt.Errorf("set %s: %w", pragma, err)
       }
   }

   db.SetMaxOpenConns(1)    // per D-08：单写入者
   db.SetMaxIdleConns(1)
   db.SetConnMaxLifetime(0) // SQLite 连接无需过期
   ```
