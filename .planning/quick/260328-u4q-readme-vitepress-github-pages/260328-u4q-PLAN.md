创建 `.github/workflows/docs.yml`，包含以下配置：

**触发条件：** push 到 main 分支且 paths 匹配 docs/** 或 .github/workflows/docs.yml，以及 workflow_dispatch 手动触发。

**权限：** contents: read, pages: write, id-token: write

**并发控制：** group: pages, cancel-in-progress: false

**build job（ubuntu-latest）：**
1. actions/checkout@v4（fetch-depth: 0）
2. pnpm/action-setup@v4（version: 10）
3. actions/setup-node@v4（node-version: 24, cache: pnpm, cache-dependency-path: docs/pnpm-lock.yaml）
4. Install dependencies: pnpm -C docs install --frozen-lockfile
5. Build docs: pnpm -C docs build
6. actions/configure-pages@v5
7. actions/upload-pages-artifact@v3（path: docs/.vitepress/dist）

**deploy job（needs: build, ubuntu-latest）：**
1. environment: name: github-pages, url 引用 deployment 输出
2. actions/deploy-pages@v4（id: deployment）
